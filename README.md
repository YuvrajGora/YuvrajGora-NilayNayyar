# DigiCampus UX Case Study

This project focuses on improving the user experience of DigiCampus based on student survey research and usability analysis.

## Research Findings

* Confusing navigation
* Poor mobile responsiveness
* Difficulty accessing study materials
* Slow loading experience

## Deliverables

* User Persona
* UX Research
* Dashboard Redesign
* Wireframes

## Tools Used

* Figma
* Google Forms
* FigJam
